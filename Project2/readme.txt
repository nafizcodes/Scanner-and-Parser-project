Necessary Precondition: 
---------------------------------------------------------------------------------------------------------------------
1. Download the Java Development Kit from, "https://www.oracle.com/java/technologies/downloads/#jdk18-windows".
2. Run the installer.
----------------------------------------------------------------------------------------------------------------------

Steps for Testing ParserProgram
---------------------------------
1. Place the ParserProgram folder from the ZIP File onto your Desktop.
2. We have included 5 text files named (testfile1.txt - testfile5.txt) for testing purposes. (If you would like to test your own text file, please place it within the ParserProgram folder at this point)
3. Open the Command Prompt on your Windows PC.
4. Inside of the Command Prompt Terminal type, "cd Desktop" (PRESS ENTER), "cd ParserProgram" (PRESS ENTER). This will place your directory location in the ParserProgram folder where your source code and textfiles exist.
5. Now, inside of the Command Prompt Terminal, type, "javac parser.java". This will compile the source code.
6. Once the source code has been compiled, then type "java parser {TEXT FILE NAME}.txt" (i.e test1.txt). => "java parser test1.txt" (PRESS ENTER).
